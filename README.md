# Landing Page Generator

A single-page Next.js app that generates a marketing landing page from a form, previews it live, and exports a fully standalone `index.html` file.

## Features

- Live landing page preview while typing
- Configurable project name, tagline, CTA, primary color, font, and 3 features
- Placeholder fallbacks for empty values
- Copy generated standalone HTML to clipboard
- Download generated HTML as `index.html`
- Copy shareable JSON config
- Local persistence with `localStorage` (safe guarded)
- Reset button to restore defaults and clear saved state

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Local Development

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Open:

- `http://localhost:3000`

## Production Build

```bash
npm run build
npm run start
```

## Environment Variables

No environment variables are required.

See `.env.example` for reference.

## Deploy to Vercel

1. Push this project to a GitHub repository.
2. Go to [vercel.com](https://vercel.com) and sign in.
3. Click **Add New... → Project**.
4. Import the GitHub repository.
5. Keep framework preset as **Next.js**.
6. No environment variables are needed.
7. Click **Deploy**.

After deployment, Vercel will provide your live URL.
